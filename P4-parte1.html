Para solucionar este ejercicio declaramos nuestras variables y las funciones para comenzar con la programacion del ejercicio:

Introducción: Funcion f[n]:

Para adaptar la función anónima f para que funcione correctamente durante operaciones de sobremuestreo como f(n/2), necesitamos interpolar los valores de la función f para los índices intermedios. Podemos hacer esto utilizando algún método de interpolación, como la interpolación lineal.

f[n] = @(n) exp(-n/5) .* cos(pi*n/5) .* (n>=0);

Codigo para definir el rango de valores y la interpolacion lineal:

f = @(n) exp(-n/5) .* cos(pi*n/5) .* (n>=0);
 
% Definir el rango de valores de n
n_values = -10:0.1:10;
 
% Computar f(n/2) utilizando interpolación lineal
f_interp = @(n) interp1(n_values, f(n_values), n/2, 'linear', 'extrap');

Calcular los valores de f(n/2) para -10 ≤ n ≤ 10:

n_half = -10:0.1:10;
f_half = f_interp(n_half);

Graficar f(n/2):

plot(n_half, f_half);
title('f(n/2)');
xlabel('n');
ylabel('f(n/2)');
