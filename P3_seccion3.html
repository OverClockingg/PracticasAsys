Introducción: Funcion en x(t)

La función x(t) se define como 0 para t < π
luego la parte impar X0(t) de la funcion x(t)
se define como x0(t) :

x0(t)= 1/2 [x(t)−x(−t)]

Utilizando esta definicion se puede escribir x0(t) en terminos
de x(t) usando cierto tiempo definido en t para capturar la forma
de la funcion, el vector definido es t = linspace(-2, 4, 1000):
t = linspace(-2, 4, 1000);
x = (2 .^ (-t)) .* cos(2 * pi * t) .* (t >= pi);

Calcular la porcion impar de x0(t):

A continuación, calcula la porción impar x0(t) de la función x(t) utilizando la definición dada, utilizamos fliplr para invertir el vector x:
x0 = 0.5 * (x - fliplr(x));
