Resuelva el problema 3.11-6.
Supongamos que existe un vector x en MATLAB espacio de trabajo, correspondiente a una duración finita Señal DT x[n].
(a) Escriba una función en MATLAB que, cuando pasa el vector x, calcula y devuelve Ex, la energía de x[n].
(b) Escriba una función en MATLAB que, cuando pasa el vector x, calcula y devuelve Px, la potencia de x[n]. Supongamos que x[n] es periódico y ese vector x contiene datos para un número entero de períodos de x[n].
(a) CALCULAR ENERGIA
%ESTA FUNCIÓN CALCULA LA ENERGIA DEL VECTOR X
function [Ex] = Energy(x)
Ex = sum(x.*conj(x));
end
(b) CALCULAR LA POTENCIA
%ESTA FUNCION CALCULA PRIMERO LA ENERGÍA Y DESPUES LA POTENCIA
function [Px] = Power(x)
% Número de muestras por período
N = length(x);
% Calculando la energía total
Ex = sum(x .* conj(x));
% Calculando la potencia
Px = Ex / N;
end
