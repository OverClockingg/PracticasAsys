Gráfica de la serie de Fourier en un intervalo que muestre 5 repeticiones:

 
% Definir el número de repeticiones a mostrar
num_repeticiones = 5;
 
% Intervalo de tiempo para mostrar 5 repeticiones de la serie de Fourier
intervalo_tiempo = linspace(-num_repeticiones*2*pi, num_repeticiones*2*pi, 1000*num_repeticiones*2);
 
% Definir la función x(t)
T = 2*pi; % Periodo de la función
a0 = 1/2; % Coeficiente a0
N = 15; % Número de armónicos
x = a0 * ones(size(intervalo_tiempo)); % Inicializar la señal x(t)
 
% Calcular la serie de Fourier
for n = 1:N
if mod(n,2) == 0
% Coeficiente an para n par (en este caso es cero)
an = 0;
else
% Coeficiente an para n impar
an = 2/(n*pi); % Coeficiente an
end
bn = 0; % Coeficiente bn (en este caso es cero)
x = x + an*cos(n*intervalo_tiempo) + bn*sin(n*intervalo_tiempo); % Sumar al término de la serie de Fourier
end
 
% Graficar la serie de Fourier
figure;
plot(intervalo_tiempo, x, 'b'); % Graficar la serie de Fourier (azul)
xlabel('Tiempo');
ylabel('Amplitud');
title('Serie de Fourier (5 repeticiones)');

Gráfica de la señal y la serie de Fourier para 15 armonicos:

% Definir el número de armónicos
N = 15;
 
% Intervalo de tiempo para mostrar la señal y la serie de Fourier
t = linspace(0, 5*2*pi, 1000); % Intervalo de 5 periodos
 
% Definir la señal original x(t)
a0 = 1/2; % Coeficiente a0
x = a0 * ones(size(t)); % Inicializar la señal x(t)
 
% Calcular la serie de Fourier
for n = 1:N
if mod(n, 2) == 0
% Coeficiente an para n par (en este caso es cero)
an = 0;
else
% Coeficiente an para n impar
an = 2/(n*pi); % Coeficiente an
end
bn = 0; % Coeficiente bn (en este caso es cero)
x = x + an*cos(n*t) + bn*sin(n*t); % Sumar al término de la serie de Fourier
end
 
% Graficar la señal original y la serie de Fourier
figure;
plot(t, x, 'b', t, a0*ones(size(t)), 'r'); % Graficar la serie de Fourier (azul) y la señal original (rojo)
xlabel('Tiempo');
ylabel('Amplitud');
title('Señal original y serie de Fourier (15 armónicos)');
legend('Serie de Fourier', 'Señal original');
