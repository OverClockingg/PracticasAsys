DISEÑO Y EVALUACIÓN DE UN FILTRO RC SIMPLE

Uno de los filtros de paso bajo más simples se realiza mediante el uso de un circuito RC, como se muestra en la Fig. 4.61
Los componentes R y C se eligen para establecer la frecuencia de corte deseada de 3 kHz. Para muchos tipos de filtros, la frecuencia de corte corresponde al punto de media potencia, o |HRC(jωc)| = 1/ √2. Asignar C a capacitancia realista de 1 nF, entonces la resistencia requerida se calcula mediante R = 1/ sqrt(C^2ωc^2) = 1/ sqrt((10−9)2(2π3000)^2).
omega_c = 2*pi*3000; C = 1e-9; R = 1/sqrt(C^2*omega_c^2)
R = 5.3052e+04
La raíz de este filtro RC de primer orden está directamente relacionada con la frecuencia de corte, λ = −1/RC = −18.850 = −ωc. Para evaluar el rendimiento del filtro RC, la respuesta de magnitud se traza en el rango de frecuencia mayormente audible (0 ≤ f ≤ 20 kHz).
f = linspace(0,20000,200); Hmag_RC = abs(CH4MP1([1],[R*C 1],f*2*pi));
plot(f,abs(f*2*pi)<=omega_c,'k-',f,Hmag_RC,'k--');
axis([0 20000 -0.05 1.05]); xlabel('f [Hz]'); ylabel('|H(j2\pi f)|');
legend('Ideal','First-order RC','location','best');

UN FILTRO RC EN CASCADA Y EXPANSIÓN POLINÓMICA

Un filtro RC de primer orden está destinado a un rendimiento deficiente; Un polo es simplemente insuficiente para obtener buenos resultados. Una cascada de circuitos RC aumenta el número de polos y mejora la respuesta del filtro. Para simplificar el análisis y evitar la carga entre etapas, empleamos seguidores de amplificador operacional para almacenar en búfer la salida de cada etapa, como se muestra en la Fig. 4.63. Una cascada de N etapas da como resultado un filtro de N-ésimo orden con una función de transferencia dada por:
Hcascade(s) = [HRC(s)]^N = (RCs+1)^−N
R = sqrt(2^(1/10)-1)/(C*omega_c)
R = 1.4213e+04
Este filtro en cascada tiene un polo de 10º orden en λ = −1/RC y no tiene ceros finitos. Para calcular la respuesta de magnitud, se necesitan los vectores de coeficiente polinómico A y B. El ajuste B = [1] garantiza que no haya ceros finitos o, de forma equivalente, que todos los ceros estén en el infinito. El comando poly, que expande un vector de raíces en un vector correspondiente de coeficientes polinómicos, se utiliza para obtener A.
B = 1; A = poly(-1/(R*C)*ones(10,1));A = A/A(end);
Hmag_cascade = abs(CH4MP1(B,A,f*2*pi));
plot(f,abs(f*2*pi)<=omega_c,'k-',f,Hmag_cascade,'k--');
axis([0 20000 -0.05 1.05]); xlabel('f [Hz]'); ylabel('|H(j2\pi f)|');
legend('Ideal','Tenth-order RC cascade','location','best');

% CH4MP2.m : Chapter 4, MATLAB Program 2
% Script M-file computes Sallen-Key component values and magnitude
% responses for each of the five cascaded second-order filter sections.
omega_0 = 3000*2*pi; % Filter cut-off frequency
psi = [9 27 45 63 81]*pi/180; % Butterworth pole angles
f = linspace(0,6000,200); % Frequency range for magnitude response calculations
Hmag_SK = zeros(5,200); % Pre-allocate array for magnitude responses
for stage = 1:5,
Q = 1/(2*cos(psi(stage))); % Compute Q for current stage
% Compute and display filter components to the screen:
disp(['Stage',num2str(stage),' (Q = ',num2str(Q),...
'): R1 = R2 = ',num2str(56000),...
', C1 = ',num2str(2*Q/(omega_0*56000)),...
', C2 = ',num2str(1/(2*Q*omega_0*56000))]);
B = omega_0^2; A = [1 omega_0/Q omega_0^2]; % Compute filter coefficients
Hmag_SK(stage,:) = abs(CH4MP1(B,A,2*pi*f)); % Compute magnitude response
end

epsilon = sqrt(10^(R/10)-1);

CP = epsilon^2*conv(CN,CN); CP(end) = CP(end)+1;
