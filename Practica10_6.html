Problema 6

Investigue la instrucción linearSystemAnalyzer (disposible a partir de 2015a) y con ayuda de esta, reporte las gráficas de
6.a Respuesta al Impulso
6.b Respuesta al escalón
6.c Respuesta a
6.d Graficas de Bode
Para condiciones iniciales cero y 10 segundos de simulación, para el sistema.
% Definición del sistema en espacio de estados
A = [0 1; -2 -3]; % Matriz de estado
B = [0; 1]; % Matriz de entrada
C = [2 0]; % Matriz de salida
D = 0; % Matriz de transmisión directa
 
sys = ss(A, B, C, D); % Crear el sistema en espacio de estados
Para el inciso a)
% 6a. Respuesta al Impulso
linearSystemAnalyzer('impulse', sys, 0:0.001:10); % Simulación de 10 segundos
Para el inciso b)
% 6b. Respuesta al Escalón
linearSystemAnalyzer('step', sys, 0:0.001:10); % Simulación de 10 segundos
Inciso c)
% 6c. Respuesta a x(t) = cos(2t)u(t)
% Definir la entrada x(t) = cos(2t)u(t)
t = linspace(0, 10, 1000); % Vector de tiempo para la simulación
x = cos(2*t) .* (t >= 0);
 
% Simulación de la respuesta del sistema
[y, t_out] = lsim(sys, x, t);
 
% Graficar entrada y salida
figure;
plot(t, x, 'LineWidth', 1.5, 'DisplayName', 'Entrada x(t) = cos(2t)u(t)');
hold on;
plot(t_out, y, 'LineWidth', 1.5, 'DisplayName', 'Respuesta del sistema');
title('Respuesta a x(t) = cos(2t)u(t)');
xlabel('Tiempo [s]');
ylabel('Amplitud');
grid on;
legend show;
hold off;
Inciso d)
% 6d. Gráficas de Bode
linearSystemAnalyzer('bode', sys); % No necesita tiempo específico
