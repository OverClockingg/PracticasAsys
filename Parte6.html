  • Gráficar señales en tiempo continuo reales, el usuario ingresa la definición de la señal en el formato especifico
  • El usuario puede definir el intervalo de la gráfica
  • El usuario puede seleccionar el tipo de transformación (horizontal, vertical o combinación) que desee realizar a la señal, la app deberá mostrar la gráfica de esta transformación.
  • Se adjunta una imagen de la app a la práctica, y el archivo de la app.

2)Programacion de los elementos:

Dado que son elementos de App designer de matlab procedere a explicar el funcionamiento que tienes cada uno con comentarios.
Esta parte del codigo funciona para obtener la funcion que ingresa el usuario :
value = app.funcion.Value;
str=split(app.funcion.Value)
senal= value;
Esta parte del codigo representa los textfiel en la cual el usuario puede ingresar valores.
Dado que matlab no puede hacer operaciones con cadena de caracteres, le decimos que lo haga de manera simbolica:
fsenal = str2sym(senal)
 
En este caso asiganamos variables para guardar los datos que el usuario ingreso y definir nuestro vector que nos servirar para graficar las señales o funciones en ese intervalo:
tn= app.tnegativo.Value
tp= app.tp0sitivo.Value
te= app.espaciado.Value
tvec=(tn:te:tp);
 
Esta parte de codigo es la mas importante pues convertimos la parte simbolica que ingreesa el usuario en una funcion de matlab en la que se le pueden ingresar valores. Primero definimos una funcion f2 para cambiar la sintaxsis de fsenal la cual es la funcion que ingresa el usuario pero solo simbolicamente.
Posteriormente la evaluamos en 1 para obtener una funcion identica pero cambiando sus propiedades, es decir; ya no es simbolica. Y la volvemos a definir como una funcion de matlab en f5:
syms x
f2= @(x) fsenal
f4= f2(1)
f5= matlabFunction(f4)
Una vez obtenida la funcion podemos operar con ella y evaluar en los valores que queramos
f6=f5(tvec)
 
f7= real(f6); % Parte real de la funcion del usuario
f8= imag(f6); % Parte imaginaria de la funcion del usuario
 
Esta parte de codigo nos ayuda a graficar los resultados, esta en un if ya que si la funcion no tiene parte imaginaria; no es necesario plotearla en mas ejes.
if f8==0
clc
plot(app.grafica,tvec,f6)
else
clc
plot3(app.grafica,tvec,f7,f8)
end
